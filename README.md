# API MyNotes - NodeJs

## O que é o MyNotes?
O MyNotes é uma pequena aplicação criada com base nos meus estudos em NodeJs, onde podem ser salvos anotações organizadas por categorias.

## Tecnologias utilizadas na API
- NodeJs - v14.15.4
- Express - v4.17.1
- Autenticação JWT
- Banco de Dados - Postgres
